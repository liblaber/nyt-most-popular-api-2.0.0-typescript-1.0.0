# MediaMetadata

**Properties**

| Name   | Type   | Required | Description                |
| :----- | :----- | :------- | :------------------------- |
| url    | string | ❌       | Image's URL.               |
| format | string | ❌       | Image's crop name.         |
| height | number | ❌       | Image's height (e.g. 293). |
| width  | number | ❌       | Image's width (e.g. 440).  |
